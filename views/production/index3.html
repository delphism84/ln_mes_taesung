Examples

Print without css styles

Col1 Col2 Col3
A B C
D E F

Code:

$("#example").printMe();

Print with external css styles

Col1 Col2 Col3
A B C
D E F

Code:

$("#example").printMe({ "path": ["path_to/eternal.css"] });

Print with title

Col1 Col2 Col3
A B C
D E F

Code:

$("#example").printMe({ "path": ["path_to/eternal.css"], "title": "Document title" });
